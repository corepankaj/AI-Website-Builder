# 🚀 AI Website Builder

AI-powered Website Builder that generates complete websites using prompts.

## ✨ Features

* 🤖 AI Website Generation (Gemini / OpenAI)
* 🧑‍💻 Monaco Code Editor
* 👀 Live Preview
* 📦 Download Website (ZIP)
* 📂 Multi-file support (HTML, CSS, JS)
* 

---

## 🛠️ Tech Stack

* Frontend: React + Tailwind CSS
* Backend: Node.js + Express
* AI: Google Gemini API
* Deployment: Vercel

---

## 📦 Installation

### 1. Clone repo

```bash
git clone https://github.com/corepankaj/AI-Website-Builder.git
cd AI-Website-Builder
```

### 2. Install dependencies

```bash
npm install
cd frontend && npm install
cd ../backend && npm install
```

### 3. Setup Environment

Create `.env` file in backend:

```env
GEMINI_API_KEY=your_api_key_here
```

### 4. Run project

```bash
# backend
cd backend
npm run dev

# frontend
cd frontend
npm start
```

---

## 🚀 Deploy

### Vercel Deployment

* Connect GitHub repo to Vercel
* Add Environment Variable:

  * `GEMINI_API_KEY`
